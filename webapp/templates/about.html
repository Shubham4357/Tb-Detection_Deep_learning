About TB Detection System

Project Overview

An advanced AI-powered system for tuberculosis detection in chest X-ray images, developed for research and educational purposes using state-of-the-art deep learning techniques.

Technical Architecture

Our system employs a sophisticated two-stage approach:

  • Stage 1 - Lung Segmentation: U-Net convolutional neural network identifies and segments lung regions from chest X-ray images, reducing noise and focusing analysis on relevant anatomical structures.
  • Stage 2 - TB Classification: DenseNet121-based classifier analyzes the segmented lung regions to detect potential tuberculosis indicators with high accuracy and reliability.

Model Training & Validation

The system has been trained using:

  • Comprehensive chest X-ray datasets with expert annotations
  • K-fold cross-validation for robust performance evaluation
  • Data augmentation techniques to improve generalization
  • Advanced metrics including accuracy, sensitivity, and specificity

Key Features

  • High Accuracy: Trained on large, diverse datasets
  • Interpretable Results: Grad-CAM visualizations for transparency
  • User-Friendly Interface: Modern web-based application
  • Research-Focused: Designed for academic and research use
